# planning_maker

How to use: <br>
1/ Create /includes/config.conf.php with the following code:
```
<?php
    define('DB_TYPE', 'mysql');
    define('DB_HOST', 'localhost');
    define('DB_NAME', 'database');
    define('DB_USER', 'user');
    define('DB_PASS', 'password');
?>
```
<br>
2/ Import struct_bdd.sql in your database
